# ucar-miniapp

二手车经纪人系统微信小程序，为经纪人提供线索查询、结算查看和数据概览等功能。

## 技术栈

- **框架**：微信原生小程序（WXML + WXSS + JS）
- **样式体系**：`unocss-preset-weapp`（原子化样式预编译）
- **图标系统**：`@unocss/preset-icons`（Lucide 图标集）
- **动画库**：内置 CSS 动效 + 微信原生 `animation` API
- **数据交互**：RESTful API（JWT 鉴权）

## 安装与运行

1. **准备工作**
   - 注册微信小程序账号
   - 获取小程序 `AppID`
   - 安装微信开发者工具

2. **导入项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择 `ucar-miniapp` 目录
   - 填写小程序 `AppID`

3. **安装依赖**
   ```bash
   cd ucar-miniapp
   npm install
   ```

4. **编译样式**
   ```bash
   npm run unocss
   ```

5. **配置API地址**
   修改 `app.js` 中的 `baseUrl`：
   ```javascript
   globalData: {
     userInfo: null,
     token: null,
     baseUrl: 'https://your-api-domain.com',
   }
   ```

6. **运行**
   - 点击「编译」按钮
   - 在模拟器中预览效果
   - 真机调试

## 功能模块

### 首页
- 数据概览（总线索、已成交、已失败、待结算金额、已结算金额）
- 快捷入口（我的线索、我的结算、个人中心、刷新数据）
- 最近线索列表（点击进入详情）
- 下拉刷新

### 线索管理
- **线索列表**
  - 7状态筛选：全部/待跟进/跟进中/已看车/已报价/已成交/已失败（横向滚动）
  - 搜索功能：按客户姓名搜索
  - 下拉刷新 + 上拉加载
  - 管理员：右下角悬浮新增按钮
  - 经纪人视角：客户电话中间4位脱敏
  - 管理员视角：额外显示归属经纪人

- **线索详情**
  - 基本信息：客户姓名/电话/类型/车辆信息/状态/归属经纪人/备注/创建时间
  - 跟进记录（仅管理员可见）：列表展示、新增、编辑、删除
  - 底部操作栏（仅管理员可见）：
    - 编辑：进入编辑表单
    - 更新状态：底部ActionSheet选择新状态（含成交价格/失败原因输入）
    - 结算/重新结算：进入结算表单
    - 删除：确认弹窗后删除

- **线索表单**
  - 新增/编辑线索
  - 字段：客户姓名、客户电话、客户类型、车辆品牌、车辆型号、归属经纪人、备注
  - 表单验证：必填项检查、手机号格式验证
  - 归属经纪人选择器：从经纪人列表中选择

### 结算管理
- **结算列表**
  - 状态筛选：全部/待结算/已结算
  - 下拉刷新 + 上拉加载
  - 管理员：右下角悬浮新增按钮
  - 点击卡片进入详情

- **结算详情**
  - 基本信息：客户姓名/车辆信息/利润金额/经纪人分成/状态/结算时间/备注/创建时间
  - 底部操作栏（仅管理员可见）：
    - 编辑：进入编辑表单
    - 更新状态：切换待结算/已结算（含结算时间选择）

- **结算表单**
  - 新增/编辑/重新结算
  - 字段：关联线索、利润金额、经纪人分成、备注
  - 表单验证：关联线索必填、利润≥0、分成≥0且≤利润
  - 线索选择：跳转线索列表选择已成交线索

### 个人中心
- 用户信息展示：微信头像、昵称、姓名、角色、状态、手机号
- 功能菜单：
  - 修改密码（含强度检测：至少8位，包含字母和数字）
  - 绑定手机号
  - 用户协议
  - 隐私政策
  - 退出登录

### 登录
- 微信一键登录
- 手机号绑定
- 协议勾选

## 技术特性

### 样式系统
- 使用 `unocss-preset-weapp` 原子化样式
- 玻璃拟态卡片（`card-glass`）
- 渐变按钮（`btn-primary`）
- 语义化标签（`tag-success`/`warning`/`danger`）
- Lucide 图标集（`i-lucide-*`）

### 权限控制
- 管理员：完整操作权限
- 经纪人：只读权限（只能查看自己的数据）
- 数据隔离：后端接口按角色返回对应数据
- 前端UI：根据角色动态显示/隐藏操作按钮

### 性能优化
- 下拉刷新 + 上拉加载
- 骨架屏加载状态
- 防抖搜索
- 图片资源优化

### 安全
- JWT 鉴权
- 手机号脱敏
- 密码强度检测
- 接口频率限制

## 注意事项

1. **微信小程序配置**
   - 确保小程序已配置合法的 `AppID`
   - 配置服务器域名（在微信公众平台）

2. **API 接口**
   - 确保后端服务已启动
   - API 接口地址配置正确
   - 接口返回格式符合规范

3. **权限配置**
   - 需要获取用户信息权限
   - 需要获取手机号权限（用于绑定）

4. **发布**
   - 提交审核前测试所有功能
   - 确保符合微信小程序审核规范
   - 配置合法的域名和隐私政策

## 开发规范

- 页面命名使用 kebab-case
- 变量命名使用 camelCase
- 常量命名使用 UPPER_SNAKE_CASE
- 代码风格保持一致
- 注释清晰
- 使用原子化样式，减少自定义CSS
- 遵循微信小程序开发最佳实践

## 常见问题

### 登录失败
- 检查微信 `AppID` 是否正确
- 检查网络连接
- 检查后端服务是否正常

### 数据不显示
- 检查 API 接口地址
- 检查登录状态
- 检查网络请求是否成功

### 样式不生效
- 运行 `npm run unocss` 重新编译样式
- 检查原子类是否正确
- 检查 `unocss.wxss` 是否被正确引入

### 下拉刷新不生效
- 检查 `enablePullDownRefresh` 配置
- 检查 `onPullDownRefresh` 实现
