# ucar-miniapp

二手车经纪人系统微信小程序，为经纪人提供线索查询、结算查看和数据概览等功能。

## 技术栈

- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 微信登录
- 网络请求封装

## 目录结构

```
ucar-miniapp/

│   └── profile/        # 个人中心

├── app.json            # 全局配置

└── README.md           # 本文件
```

## 安装与运行

1. **准备工作**
   - 注册微信小程序账号
   - 获取小程序 `AppID`
   - 安装微信开发者工具

2. **导入项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择 `ucar-miniapp` 目录
   - 填写小程序 `AppID`

3. **配置API地址**
   修改 `utils/request.js` 中的 `BASE_URL`：
   ```javascript
   const BASE_URL = 'https://your-api-domain.com/api/v1'
   ```

4. **运行**
   - 点击「编译」按钮
   - 在模拟器中预览效果
   - 真机调试

### 结算页
- 结算列表展示
- 状态筛选（全部、待结算、已结算）

- 上拉加载

### 个人中心

- 微信头像和昵称
- 退出登录

## 登录流程

1. **微信登录**
   - 调用 `wx.login()` 获取 code
   - 后端验证 code 获取 openid
   - 检查是否已绑定手机号

2. **手机号绑定**
   - 未绑定手机号时跳转到绑定页面
   - 输入手机号完成绑定
   - 生成 JWT token

3. **登录状态维护**
   - token 存储在本地存储
   - 每次请求自动携带 token
   - 登录过期自动跳转登录

## 注意事项

1. **微信小程序配置**
   - 确保小程序已配置合法的 `AppID`
   - 配置服务器域名（在微信公众平台）

2. **API 接口**
   - 确保后端服务已启动
   - API 接口地址配置正确
   - 接口返回格式符合规范

3. **权限配置**
   - 需要获取用户信息权限
   - 需要获取手机号权限（用于绑定）

4. **性能优化**
   - 图片资源优化
   - 页面渲染性能
   - 网络请求优化

5. **发布**
   - 提交审核前测试所有功能
   - 确保符合微信小程序审核规范
   - 配置合法的域名和隐私政策

## 常见问题

### 登录失败
- 检查微信 `AppID` 是否正确
- 检查网络连接
- 检查后端服务是否正常

### 数据不显示
- 检查 API 接口地址
- 检查登录状态
- 检查网络请求是否成功

### 下拉刷新不生效
- 检查 `enablePullDownRefresh` 配置
- 检查 `onPullDownRefresh` 实现
