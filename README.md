# 二手车收车经纪人系统

## 项目简介

二手车收车经纪人系统是一个为汽车经纪人提供线索管理、结算管理等功能的系统，包括PC端管理后台和微信小程序端。

### 项目目标
- 管理员：规范线索管理、结算流程，避免遗漏
- 经纪人：透明化收益，实时结算提醒，减少沟通成本

### 核心角色
- **管理员**：后台操作，负责线索管理、结算管理、用户管理
- **经纪人**：微信小程序操作，负责查看线索、查看结算、接收结算提醒

## 技术栈

- **后端**：Node.js + Express（统一服务，供管理后台和小程序使用）
- **PC端前端（管理后台）**：Vue3
- **小程序**：微信小程序原生框架
- **数据库**：MySQL 8+

## 主要功能

### 管理后台功能
- **登录**：账号密码+验证码登录
- **线索管理**：新增线索、查看线索列表、修改线索状态、查看线索详情
- **结算管理**：新增结算、查看结算列表、更新结算状态、推送结算提醒
- **用户管理**：新增经纪人、查看经纪人列表、修改经纪人信息、管理经纪人状态

### 微信小程序功能
- **登录**：微信登录
- **首页概览**：总线索数、成交数、失败数、待结算金额、已结算金额
- **线索查看**：查看本人线索列表、线索详情
- **结算查看**：查看待结算和已结算列表、累计收益统计
- **结算提醒**：接收微信服务通知
- **个人中心**：展示个人信息、退出登录

## 交互逻辑
- 管理员登录（账号密码+验证码）→ 录入线索 → 状态流转 → 成交 → 结算 → 推送提醒 → 经纪人查看收益
- 经纪人微信登录 → 查看线索和结算信息

## 数据库设计

系统包含以下数据表：
- `c_user`：用户表（管理员和经纪人）
- `c_lead`：线索表
- `c_settlement`：结算表
- `c_operation_log`：操作日志表
- `c_dict`：码表/字典表

详细的数据库设计请参考 `doc/3. database-schema.md`。

## 开发规范

- **命名规范**：
  - 文件夹：小写-中划线
  - 变量：camelCase
  - 数据表：snake_case

- **接口路径**：统一使用 `/api/v1/` 前缀

- **提交规范**：
  - 格式：`类型(范围): 描述`
  - 类型：feat, fix, docs, style, refactor, test, chore
  - 范围：模块名称
  - 描述：简洁明了

- **代码风格**：
  - ESLint + Prettier
  - 2空格缩进
  - 单引号

- **错误处理**：
  - 统一的JSON返回格式
  - 错误码定义

- **安全规则**：
  - JWT鉴权
  - 接口分离

- **性能约定**：
  - 接口响应时间
  - 数据库索引

- **部署规则**：
  - Docker部署
  - 日志管理

- **API版本管理**：
  - 统一前缀

- **测试与质量保证**：
  - 单元测试覆盖率
  - ESLint检查

- **文档维护规则**：
  - 当数据表结构发生变化时，必须更新 `doc/3. database-schema.md` 文件
  - 当码表（c_dict）数据发生变化时，必须更新 `sql/初始化码表.sql` 文件
  - 当数据表结构或业务逻辑发生变化时，必须更新 `sql/常用sql.sql` 文件
  - 所有文档修改都应该保持一致性，确保不同文档之间的信息不冲突
  - 文档更新后，相关的SQL文件也应该同步更新
  - 新增功能或修改功能时，应同时更新相关的文档

详细的开发规范请参考 `doc/2. rules.md`。

## 安装和运行

### 环境要求
- Node.js v24.12.0
- MySQL 8+
- 微信开发者工具（用于小程序调试）
- Docker（推荐，用于后端服务部署）
- Nginx（用于前端静态资源部署）

### 环境变量
- DB_HOST
- DB_USER
- DB_PASS
- DB_NAME
- WX_APPID
- WX_SECRET
- JWT_SECRET

### 后端安装

1. 进入后端目录（ucar-api）
2. 安装依赖：`npm install`
3. 配置环境变量（复制 `.env.example` 为 `.env` 并修改）
4. 启动服务：`npm start`

### 前端安装

1. 进入前端目录（ucar-admin-web）
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 构建生产版本：`npm run build`

### 小程序安装

1. 进入小程序目录（ucar-miniapp）
2. 使用微信开发者工具打开项目
3. 安装依赖：`npm install`
4. 构建项目：`npm run build`
5. 预览或发布小程序
