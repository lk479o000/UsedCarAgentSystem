# 二手车经纪人管理系统 - 设计系统融合版 v1.0

## 1. 设计理念
- **专业可靠**：深蓝灰与明亮蓝结合，兼顾稳重与活力
- **现代简洁**：简洁线条与布局，减少视觉干扰
- **数据驱动**：清晰的数据可视化，突出关键信息
- **高效操作**：优化工作流程，减少操作步骤
- **跨平台一致**：后台与微信小程序保持统一风格
- **可扩展性**：支持未来功能扩展与模块化迭代
- **玻璃拟态与深度**：引入玻璃拟态与分层阴影，提升界面质感
- **动态色彩**：支持根据用户偏好或品牌调性自动调整色彩
- **微交互优先**：每个交互元素都有细腻反馈

## 2. 颜色方案
### 主色调
- **primary**: #165DFF（明亮蓝） + #1e293b（深蓝灰）
- **primary-light**: #4080FF / #334155
- **primary-dark**: #0E42CC / #0f172a
- **primary-gradient**: linear-gradient(135deg, #165DFF 0%, #4080FF 100%)

### 功能色
- **success**: #10b981（绿色）
- **warning**: #f59e0b（橙色）
- **danger**: #dc2626（红色）
- **info**: #3b82f6（蓝色）

### 中性色
- **background**: #f8fafc（浅灰白）
- **surface**: #ffffff（白色）
- **text-primary**: #1e293b（深蓝灰）
- **text-secondary**: #64748b（浅蓝灰）
- **border**: #e2e8f0（浅灰）

### 高亮色
- **highlight**: #9333ea（紫色）
- **accent**: #f43f5e（玫瑰红）

### 暗色模式
- **background**: #0f172a
- **surface**: #1e293b
- **text-primary**: #f8fafc
- **text-secondary**: #cbd5e1

## 3. 排版系统
- **主字体**: 中文优先（苹方/思源黑体），英文数字用 Inter/Roboto
- **代码字体**: JetBrains Mono / Fira Code
- **字号层次**:
  - H1: 32px / 700 / 40px
  - H2: 24px / 600 / 32px
  - H3: 20px / 600 / 28px
  - H4: 16-18px / 500 / 24px
  - Body: 14px / 400 / 22px
  - Small: 12px / 400 / 18px
  - Caption: 11px / 400 / 16px
- **行高**: 正文 1.6倍，标题 1.3倍
- **字间距**: 标题 -0.02em，英文大写 0.05em

## 4. 间距与圆角系统
- **间距**: 4px, 8px, 16px, 24px, 32px, 48px
- **圆角**: 8px（常用），12px（卡片），16px（对话框），full（头像/按钮）

## 5. 阴影与深度
- **shadow-sm**: 轻微阴影
- **shadow-md**: 中等阴影
- **shadow-lg**: 大型阴影
- **shadow-glow**: 发光效果
- **glass**: 背景半透明 + blur(12px)

## 6. 组件库
- **按钮**: 渐变主色，圆角8px，hover阴影增强
- **卡片**: 白色背景 + 阴影，支持玻璃拟态
- **表格**: 斑马纹，支持排序、展开、紧凑/宽松模式
- **表单**: 输入框聚焦 glow 效果，选择器/日期选择器带动画
- **对话框**: 标准/确认/信息/抽屉/全屏，圆角16px，阴影xl
- **通知**: 成功/错误/警告/信息，右上角堆叠显示
- **标签**: 标准/警告/信息/成功/危险/幽灵，支持可删除、状态指示

## 7. 布局系统
- **后台**: 顶部导航64px，侧边导航240px可折叠，主内容区32px内边距
- **小程序**: 顶部导航44px，底部导航60px，卡片间距16px，安全区适配

## 8. 数据可视化
- **图表类型**: 折线图、柱状图、饼图、环形图、仪表盘、热力图、散点图、雷达图、漏斗图
- **风格**: 主色渐变，圆角柱子，交互提示框，支持暗色模式
- **交互**: 悬停显示数据，下钻分析，缩放/平移

## 9. 交互动效
- **基础动画**: 淡入淡出、滑动、缩放、旋转、弹跳
- **微交互**: 按钮缩放反馈，输入框 glow，卡片悬停阴影增强
- **页面转场**: 淡入、右侧滑入、中心缩放
- **加载状态**: 骨架屏、进度条、环形进度、空状态插画

## 10. 响应式设计
- **后台**: ≥1440px 多列，1200-1439px 调整列数，<768px 单列抽屉式
- **小程序**: 竖屏单列，横屏双列，适配刘海屏/底部安全区

## 11. 最佳实践
- **可用性**: 操作不超过3次点击，错误提示清晰，反馈及时
- **性能**: 图片WebP，懒加载，虚拟滚动，缓存策略
- **可访问性**: WCAG AA对比度，键盘导航，屏幕阅读器兼容
- **安全性**: 敏感数据脱敏显示，权限矩阵，操作日志

## 12. 微信小程序优化
- **视觉**: 深蓝主色，红色CTA，卡片圆角12px，统一图标系统
- **性能**: 分包加载，骨架屏，虚拟列表，防抖节流
- **交互**: 下拉刷新，上拉加载，左滑操作，触觉反馈
- **个性化**: 首页快捷入口，智能提醒，自定义布局
