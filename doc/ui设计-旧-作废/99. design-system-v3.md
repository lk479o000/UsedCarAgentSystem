# 二手车经纪人管理系统 - 设计系统融合版 v3.0

## 1. 设计理念
- **温暖可靠**：米色与柔和灰为基底，传达亲和与信任
- **现代极简**：大字号标题，轻量正文，减少视觉噪音
- **数据驱动**：清晰的数据可视化，支持下钻与实时更新
- **高效操作**：操作不超过3次点击，微交互反馈及时
- **跨平台一致**：后台与微信小程序保持统一风格
- **玻璃拟态 + 新拟态**：卡片与面板采用玻璃质感与柔和阴影
- **动态色彩系统**：支持根据用户偏好或品牌自动调整色彩
- **沉浸式暗色模式**：霓虹点缀，提升未来感

## 3. 排版系统
- **主字体**: 中文（苹方/思源黑体），英文数字用 Inter/Roboto
- **字号层次**:
  - H1: 36px / 700 / 44px
  - H2: 28px / 600 / 36px
  - H3: 20px / 600 / 28px
  - Body: 16px / 400 / 24px
  - Small: 13px / 400 / 20px
- **字间距**: 标题 -0.02em，正文 0，英文大写 0.05em

## 4. 间距与圆角系统
- **间距**: 8px, 16px, 24px, 32px, 48px
- **圆角**: 12px（常用），16px（卡片），24px（对话框），full（头像/按钮）

## 5. 阴影与深度
- **shadow-sm**: 0 1px 2px rgba(0,0,0,0.05)
- **shadow-md**: 0 4px 6px rgba(0,0,0,0.1)
- **shadow-lg**: 0 10px 15px rgba(0,0,0,0.15)
- **glass**: backdrop-filter: blur(12px), 半透明背景
- **neumorphism**: 内阴影 + 柔和外阴影，提升触感

## 6. 组件库
- **按钮**: 琥珀渐变背景，圆角12px，hover阴影增强，点击缩放反馈
- **卡片**: 米白背景 + 玻璃拟态，圆角16px，柔和阴影
- **表格**: 斑马纹，支持排序、展开，下钻分析
- **表单**: 输入框 glow 效果，选择器/日期选择器带动画
- **对话框**: 标准/确认/信息/抽屉/全屏，圆角16-24px，玻璃质感
- **通知**: 成功/错误/警告/信息，右上角堆叠显示，带弹跳动画
- **标签**: 状态标签带圆点，支持可删除、幽灵样式

## 7. 布局系统
- **后台**: 顶部导航64px，侧边导航240px可折叠，主内容区32px内边距
- **小程序**: 顶部导航44px，底部导航60px，卡片间距16px，适配安全区

## 8. 数据可视化
- **图表类型**: 折线、柱状、饼图、环形、仪表盘、热力图、散点、漏斗、堆叠柱状
- **风格**: 琥珀渐变，圆角柱子，玻璃拟态提示框，支持暗色模式
- **交互**: 悬停显示数据，下钻分析，缩放/平移，实时更新动画

## 9. 交互动效
- **基础动画**: 淡入淡出、滑动、缩放、旋转、弹跳
- **微交互**: 按钮缩放反馈，输入框 glow，卡片悬停阴影增强
- **页面转场**: 淡入、右侧滑入、中心缩放
- **加载状态**: 骨架屏、渐变进度条、环形进度、空状态插画

## 10. 响应式设计
- **后台**: ≥1440px 多列，1200-1439px 调整列数，<768px 单列抽屉式
- **小程序**: 竖屏单列，横屏双列，适配刘海屏/底部安全区

## 11. 最佳实践
- **可用性**: 操作不超过3次点击，错误提示清晰，反馈及时
- **性能**: 图片WebP，懒加载，虚拟滚动，缓存策略
- **可访问性**: WCAG AA对比度，键盘导航，屏幕阅读器兼容
- **安全性**: 敏感数据脱敏显示，权限矩阵，操作日志

## 12. 微信小程序优化
- **视觉**: 琥珀橙主色 + 荧光绿强调，卡片圆角16px，统一图标系统
- **性能**: 分包加载，骨架屏，虚拟列表，防抖节流
- **交互**: 下拉刷新，上拉加载，左滑操作，触觉反馈
- **个性化**: 首页快捷入口，智能提醒，自定义布局
