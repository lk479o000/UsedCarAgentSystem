# 二手车收车经纪人系统 - 需求说明书

## 项目目标
- 管理员：规范线索管理、结算流程，避免遗漏
- 经纪人：透明化收益，实时结算提醒，减少沟通成本

## 核心角色
- 用户（通过role字段区分：0=管理员, 1=经纪人）
  - 管理员：微信小程序操作 + PC后台操作，负责线索管理、结算管理、用户管理
  - 经纪人：微信小程序操作 + PC后台操作，负责查看线索、查看结算、接收结算提醒

## 功能需求
- 后台：登录（账号密码+验证码）、线索管理、结算管理、用户管理
- 小程序：登录（微信登录）、首页概览、线索查看、结算查看、结算提醒、个人中心

## 交互逻辑
- 管理员登录（账号密码+验证码）→ 录入线索 → 状态流转 → 成交 → 结算 → 推送提醒 → 经纪人查看收益
- 经纪人微信登录 → 查看线索和结算信息
