# 测试用例设计

## 登录
- **PC端**：
  - 输入正确管理员账号+密码+验证码 → 成功登录，返回 token
  - 输入错误密码 → 登录失败，返回错误码
  - 验证码错误/过期 → 登录失败，返回错误码
  - 连续 5 次密码错误 → 账号锁定 30 分钟
  - 经纪人使用账号密码登录 → 成功返回 token
  - 禁用状态的用户账号密码登录 → 返回账号已禁用错误
- **微信端**：
  - 管理员微信登录 → 成功返回 token
  - 已绑定手机号的经纪人微信登录 → 直接返回 token
  - 未绑定手机号的用户微信登录 → 创建临时用户，返回 need_bind_phone=true
  - 绑定手机号时手机号已存在且未绑定微信 → 将微信关联到该手机号用户，返回 token
  - 绑定手机号时手机号不存在但微信已存在（临时用户） → 更新手机号，返回 token
  - 绑定手机号时手机号和微信都存在且为同一用户 → 返回成功
  - 绑定手机号时手机号和微信都存在但为不同用户 → 返回错误码 18
  - 绑定手机号时手机号和微信都不存在 → 创建新用户，返回 token
  - 绑定手机号时手机号格式错误 → 返回错误码 21
  - 禁用状态的用户微信登录 → 返回账号已禁用错误

## 线索管理（管理员）
- **功能测试**：
  - 新增线索 → 列表显示新增数据
  - 新增线索时客户手机号已存在 → 返回错误提示（是否允许重复需根据业务确认）
  - 编辑线索 → 成功更新数据
  - 删除线索 → 软删除，is_deleted=1，关联结算保留
  - 尝试删除已成交线索 → 返回错误（已成交不可删除）
- **状态流转**：
  - 修改状态：待跟进(0) → 跟进中(1) → 已看车(2) → 已报价(3) → 已成交(4) → 成功
  - 修改状态为已成交(4) 时未提供 car_actual_price → 返回错误码 11
  - 修改状态为已失败(5) 时未提供 fail_reason → 返回错误码 11
  - 修改状态：已成交(4) → 已失败(5) → 返回错误码 12（终态不可修改）
  - 修改状态：已报价(3) → 待跟进(0) → 返回错误码 10（状态流转非法）
  - 状态改为已成交(4) 后 → 自动创建待结算记录
- **列表查询**：
  - 查询线索列表 → 支持分页与筛选，page/size 超出限制时按最大值处理
  - 搜索功能 → 按客户姓名搜索，结果正确
- **跟进记录**：
  - 新增跟进记录 → 成功添加到线索详情
  - 编辑跟进记录 → 成功更新内容
  - 删除跟进记录 → 成功从列表中移除

## 结算管理（管理员）
- **功能测试**：
  - 成交线索 → 进入待结算列表
  - 新增结算时 lead_id 对应线索未成交 → 返回错误码 13
  - 新增结算时 agent_share > profit → 返回错误码 14
  - 新增结算时该线索已有结算记录 → 返回错误码 15

  - 已结算记录尝试修改 profit/agent_share → 返回错误码 16

- **提醒功能**：
  - 5 分钟内重复推送同一结算提醒 → 返回错误码 17
- **统计与导出**：
  - 查看统计 → 数据正确（总利润、分成金额）
  - 导出结算列表 → 生成 Excel 文件，字段完整
- **边界值**：
  - 结算金额为负数/零 → 返回错误提示

## 安全测试
- **鉴权测试**：
  - 未登录状态访问接口 → 返回鉴权失败 (code=2)
  - 使用过期/伪造 token 访问接口 → 返回鉴权失败
  - 经纪人尝试访问管理员接口 → 返回越权错误
  - 管理员尝试访问经纪人专用接口 → 返回错误提示
- **防攻击测试**：
  - 接口频率限制测试：同一 IP 快速请求登录接口 → 返回 429
  - SQL 注入测试：在查询参数中传入特殊字符 → 正常过滤，不报错
  - XSS 测试：在备注字段输入脚本标签 → 存储时转义，输出时安全
- **数据安全**：
  - 密码存储 → 使用 bcrypt 加密
  - 敏感信息 → 手机号脱敏显示
  - 环境变量 → 数据库连接信息不硬编码

## 边界值测试
- **输入验证**：
  - 手机号格式校验：11位数字、非11位、包含字母、空值
  - 金额字段：最大值、0、负数、超大值
  - 字符串字段：空值、最大长度、超长（超过数据库限制）
  - 状态字段：非法值（如 99、-1、字符串）
- **分页参数**：
  - page=0、page=-1、size=0、size=1000、size=-10
- **文件操作**：
  - Excel 导入：空文件、格式错误、超大文件（>10MB）

## 异常场景测试
- **系统异常**：
  - 数据库连接断开 → 返回系统错误 (code=4)，不暴露堆栈信息
  - 微信服务不可用 → 微信登录返回友好错误提示
  - 文件导出时数据量过大 → 异步生成，提供下载链接
  - Excel 导入时列名不匹配 → 返回清晰的错误提示
- **用户操作**：
  - 网络中断 → 显示网络错误提示
  - 操作超时 → 显示超时提示
  - 重复提交 → 防止重复操作，显示提示
