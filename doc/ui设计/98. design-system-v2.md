# 二手车经纪人管理系统 - 设计系统 v2.1

## 1. 设计理念

- **专业可靠**：采用深色蓝色系为主色调，传达汽车行业的专业感和信任感
- **现代简洁**：使用简洁的线条和布局，减少视觉干扰
- **数据驱动**：清晰的数据可视化，突出关键信息
- **高效操作**：优化工作流程，减少操作步骤
- **跨平台一致**：确保管理后台和微信小程序的设计风格统一
- **可扩展性**：支持未来功能扩展与模块化迭代
- **玻璃拟态与深度**：引入现代玻璃拟态（Glassmorphism）和分层深度设计，提升界面质感
- **动态色彩**：支持动态色彩系统，根据用户偏好或品牌调性自动调整色彩方案
- **微交互优先**：每个交互元素都应有细腻的反馈，提升用户操作愉悦感

## 9. 数据可视化

- **折线图**: 展示趋势数据，如线索数量变化，支持多线对比和面积填充
- **柱状图**: 展示对比数据，如不同经纪人的业绩，支持圆角柱子和渐变填充
- **饼图/环形图**: 展示占比数据，如线索状态分布，支持中心文字和图例交互

- **漏斗图**: 展示转化流程，如线索到成交的转化漏斗

## 10. 交互动效

### 基础动画
- **淡入淡出**: 页面切换、组件显示/隐藏，时长300ms，ease-in-out
- **滑动**: 侧边栏展开/收起、对话框弹出，时长300ms，cubic-bezier(0.4, 0, 0.2, 1)
- **缩放**: 卡片悬停、按钮点击，时长200ms，ease-out
- **旋转**: 加载状态、刷新按钮，时长1s，linear，无限循环
- **弹跳**: 通知出现、重要提示，时长400ms，cubic-bezier(0.68, -0.55, 0.265, 1.55)

### 微交互
- **按钮反馈**: 点击时轻微缩放(0.97)，hover时阴影增强并上移2px
- **输入框反馈**: 聚焦时边框颜色变化，带阴影glow，标签上浮动画
- **表格行反馈**: 悬停时背景色变化，带过渡动画
- **通知反馈**: 滑入/滑出动画，带透明度变化
- **卡片反馈**: 悬停时阴影增强，轻微上移，带过渡动画
- **图标反馈**: 悬停时图标颜色变化，带旋转或缩放效果

### 页面转场
- **淡入**: 新页面淡入，时长300ms
- **滑动进入**: 从右侧滑入，时长300ms，用于详情页
- **缩放进入**: 从中心缩放进入，时长300ms，用于模态框

## 13. 设计最佳实践

### 可用性
- **操作简化**: 所有操作不超过3次点击
- **错误提示**: 明确的错误信息和引导，带图标和颜色标识
- **反馈及时**: 操作后立即给出视觉反馈，加载状态明确
- **导航清晰**: 明确的面包屑和导航路径，当前位置高亮
- **一致性**: 相同功能使用相同图标和文案

### 性能
- **图片优化**: 适当压缩和懒加载，使用WebP格式
- **代码分割**: 按需加载组件，减少首屏加载时间
- **动画性能**: 使用CSS3动画和transform，避免触发重排
- **缓存策略**: 合理使用浏览器缓存和本地存储
- **虚拟滚动**: 长列表使用虚拟滚动，减少DOM节点

### 可访问性
- **色彩对比度**: 符合WCAG AA标准，文字与背景对比度至少4.5:1
- **键盘导航**: 支持Tab键导航，焦点状态明显
- **屏幕阅读器**: 兼容屏幕阅读器，提供ARIA标签
- **语义化HTML**: 使用正确的HTML标签，提供结构化数据
- **焦点管理**: 模态框和对话框打开时捕获焦点，关闭时恢复焦点
