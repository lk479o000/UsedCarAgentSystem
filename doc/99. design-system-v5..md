# 微信小程序开发文档（基于现代美学与设计系统 v4.0）

## 🎯 项目目标
打造与 PC 端统一的现代化小程序界面，采用 **玻璃拟态 + 霓虹蓝点缀** 风格，兼顾科技感与操作效率。

---

## 🧩 技术栈与插件
- **框架**：微信原生小程序（WXML + WXSS + JS）
- **样式体系**：`unocss-preset-weapp`（原子化样式预编译）
- **图标系统**：`@unocss/preset-icons`（Lucide 图标集）
- **动画库**：内置 CSS 动效 + 微信原生 `animation` API
- **数据交互**：RESTful API（JWT 鉴权）
- **性能优化**：分包加载 + 骨架屏 + 虚拟列表

# 微信小程序开发规范补充说明

## 样式编译说明
由于项目是纯微信原生小程序，无法直接使用 `i-lucide-*` 格式的图标或 UnoCSS 的运行时功能。  
为实现设计文档中的原子化样式与图标需求，需要通过 **unocss-preset-weapp** 将 WXML 中的 class 编译为 WXSS 文件。

### 编译流程
1. **安装依赖**
  ```bash
  npm install -D unocss unocss-preset-weapp @unocss/preset-icons
  ```
2. 配置文件 `ocss.config.js`
  ```js
  import { defineConfig } from 'unocss'
  import presetWeapp from 'unocss-preset-weapp'
  import { transformerClass } from 'unocss-preset-weapp/transformer'

  const include = [/\.wxml$/]

  export default defineConfig({
    content: { pipeline: { include } },
    presets: [presetWeapp()],
    transformers: [transformerClass({ include, classTags: false })],
    separators: '__', // hover__bg-blue-500
  })
  ```
3. 编译命令
  ```bash
  npm run unocss
  ```
- 输出文件：`unocss.wxss`
- 在 `app.wxss` 中引入：
  ```css
  @import './unocss.wxss';
  ```

### 使用说明
- 开发时只需在 WXML 写原子类（如 `flex p-4 bg-primary`），WXSS 会自动生成。
- 图标可通过 `@unocss/preset-icons` 使用，例如：
  ```html
  <view class="i-lucide-list"></view>
  ```
- 所有样式和图标必须依赖 unocss-preset-weapp 编译，否则无法生效。
