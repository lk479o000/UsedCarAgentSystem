# 部署说明

## 环境要求
- Node.js v24.12.0
- MySQL 8+
- 微信开发者工具（用于小程序调试）
- Docker（推荐，用于后端服务部署）
- Nginx（用于前端静态资源部署）

---

## 环境变量
- DB_HOST
- DB_USER
- DB_PASS
- DB_NAME
- WX_APPID
- WX_SECRET
- JWT_SECRET

---

## 部署方式

### 后端（ucar-api）
- 使用 Docker 部署：
```bash
docker build -t ucar-api .
docker run -d -p 3000:3000 --env-file .env ucar-api
```

### 管理后台前端（ucar-admin-web）
- 构建：
```bash
npm run build
```

- 部署到 Nginx：
```nginx
server {
  listen 80;
  server_name admin.example.com;
  root /var/www/ucar-admin-web/dist;
  index index.html;
}
```

### 微信小程序（ucar-miniapp）
- 导入微信开发者工具
- 配置 AppID 与服务器地址
- 上传审核并发布

---

## 日志与监控
- 后端日志：按日期切割存储在 `/logs`
- 使用 PM2 或 Docker logs 查看运行日志
- 推荐接入轻量化监控（如 Prometheus + Grafana）

## 安全配置
- 所有接口必须通过 HTTPS
- 配置跨域（CORS），仅允许指定域名访问
- 数据库账号需最小权限原则

---

## 数据库备份与恢复
- 每日自动备份数据库到 `/backup`
- 仅保留最近 7 天备份
- 恢复方式：
```bash
mysql -u user -p db_name < backup.sql
```

---

## 多环境说明
- 开发环境：本地运行，使用测试数据库
- 测试环境：Docker 部署，测试服务器
- 生产环境：Docker + Nginx，正式域名，开启 HTTPS
