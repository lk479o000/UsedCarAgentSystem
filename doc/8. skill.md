# AI Skill 模块分工说明

## 总体原则
- 每个 skill 对应一个开发职责模块
- 输入：文档或需求说明
- 输出：代码、配置或文档片段
- 所有 skill 可独立运行，也可协同工作

---

## Skill 列表

### ui-ux-pro-max
- **职责**：负责界面设计与交互逻辑（基于Vue3）
- **输入**：`ui-ux.md`
- **输出**：Vue3前端页面代码、交互说明、样式文件
- **协作**：与 `api-builder` 对接接口，与 `test-genius` 验证交互流程

### db-master
- **职责**：负责数据库建模与 SQL 生成
- **输入**：`database-schema.md`
- **输出**：建表语句、索引优化脚本
- **协作**：与 `api-builder` 对接数据层

### api-builder
- **职责**：负责接口设计与实现
- **输入**：`api-spec.md`
- **输出**：后端接口代码、鉴权逻辑
- **协作**：与 `db-master` 共享数据结构，与 `ui-ux-pro-max` 和 `ucar-miniapp-dev` 提供接口

### test-genius
- **职责**：负责测试用例与自动化脚本
- **输入**：`test-case.md`
- **输出**：单元测试、接口测试脚本
- **协作**：与所有 skill 验证功能正确性

### deploy-helper
- **职责**：负责部署脚本与环境配置
- **输入**：`deploy.md`
- **输出**：Dockerfile、Nginx 配置、环境变量模板
- **协作**：与 `api-builder` 部署后端，与 `ui-ux-pro-max` 部署前端，与 `ucar-miniapp-dev` 配置小程序部署

### ucar-miniapp-dev
- **职责**：负责微信小程序开发
- **输入**：`ui-ux.md`、`api-spec.md`、`9. 小程序功能规划方案.md`
- **输出**：小程序页面代码、样式文件、配置文件
- **协作**：与 `api-builder` 对接接口，与 `test-genius` 验证小程序功能

## 扩展技能

### log-watcher
- **职责**：日志监控与告警
- **输入**：运行日志
- **输出**：告警通知、日志分析报告

### perf-optimizer
- **职责**：性能优化与压力测试
- **输入**：`test-case.md` + 系统运行数据
- **输出**：优化建议、索引调整、缓存策略

### doc-writer
- **职责**：生成开发文档与使用手册
- **输入**：所有 `.md` 文件
- **输出**：统一的 README、API 文档、用户手册

---

## 优先级说明
- **初期必备**：`db-master`、`api-builder`、`ui-ux-pro-max`
- **中期必备**：`test-genius`、`deploy-helper`、`ucar-miniapp-dev`
- **后期优化**：`log-watcher`、`perf-optimizer`、`doc-writer`

---

## 技能模块协作流程

### 开发流程
1. **需求分析**：基于 `1. 需求说明书.md` 和 `9. 小程序功能规划方案.md`
2. **数据库设计**：`db-master` 根据需求设计数据库结构
3. **接口开发**：`api-builder` 实现后端接口
4. **前端开发**：
   - `ui-ux-pro-max` 开发管理后台前端
   - `ucar-miniapp-dev` 开发微信小程序
5. **测试验证**：`test-genius` 编写并执行测试用例
6. **部署上线**：`deploy-helper` 配置部署环境
7. **监控优化**：`log-watcher` 和 `perf-optimizer` 进行系统监控和优化

### 文档更新流程
- 所有功能变更必须同步更新相关文档
- `doc-writer` 负责整合所有文档，确保一致性
- 数据库变更 → 更新 `database-schema.md`
- 接口变更 → 更新 `api-spec.md`
- 功能变更 → 更新 `test-case.md`
- 小程序功能变更 → 更新 `9. 小程序功能规划方案.md` 和 `9. 小程序功能开发进度.md`

## 技能模块能力要求

### ucar-miniapp-dev 能力要求
- **技术栈**：微信原生小程序框架、unocss-preset-weapp、Lucide 图标
- **功能开发**：
  - 小程序页面结构设计与实现
  - 原子化样式应用
  - 微信登录与手机号绑定
  - 权限控制（管理员/经纪人视角）
  - 数据交互与状态管理
- **性能优化**：
  - 小程序启动速度优化
  - 页面渲染性能优化
  - 网络请求优化
- **发布流程**：
  - 小程序审核规范
  - 版本管理
  - 部署配置

### 其他技能模块能力要求
- **ui-ux-pro-max**：Vue3、Tailwind CSS、响应式设计
- **api-builder**：Node.js、Express、JWT 鉴权
- **db-master**：MySQL、SQL 优化、数据库设计
- **test-genius**：单元测试、接口测试、性能测试
- **deploy-helper**：Docker、Nginx、环境配置
