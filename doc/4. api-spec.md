# 接口设计规范

## 通用说明
- 所有接口统一前缀：`/api/v1`
- 所有接口返回统一 JSON 格式：
```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```
- 鉴权方式：JWT，登录成功后返回 token，后续请求需在 Header 中携带：`Authorization: Bearer <token>`

## 登录接口

### 获取验证码
- GET `/api/v1/auth/captcha`
- 功能：获取登录验证码
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "captcha_id": "xxxx",
    "captcha_image": "data:image/png;base64,xxxx"
  }
}
```

### 管理员登录
- POST `/api/v1/auth/login`
- 请求参数：
```json
{
  "username": "admin",
  "password": "123456",
  "captcha": "1234",
  "captcha_id": "xxxx"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "xxxx",
    "user_info": {
      "id": 1,
      "name": "管理员",
      "role": 0,
      "status": 1
    }
  }
}
```

### 修改密码
- PUT `/api/v1/auth/password`
- 请求参数：
```json
{
  "old_password": "123456",
  "new_password": "654321",
  "new_password_confirm": "654321"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success"
}
```

### 设置安全密码
- PUT `/api/v1/auth/secure-password`
- 请求参数：
```json
{
  "secure_password": "your-secure-password"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success"
}
```

### 经纪人微信登录
- POST `/api/v1/auth/login/wechat`
- 功能：使用微信公众号或小程序登录
- 请求参数：
```json
{
  "open_id": "xxxx",
  "nickname": "张三",
  "avatar_url": "https://example.com/avatar.jpg"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "xxxx",
    "user_info": {
      "id": 1,
      "name": "张三",
      "nickname": "张三",
      "phone": "13800000000",
      "role": 1,
      "status": 1
    }
  }
}
```

### 绑定手机号
- POST `/api/v1/auth/bind-phone`
- 功能：微信登录后绑定手机号
- 请求参数：
```json
{
  "open_id": "xxxx",
  "phone": "13800000000",
  "code": "1234"  # 短信验证码
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "xxxx",
    "user_info": {
      "id": 1,
      "name": "张三",
      "nickname": "张三",
      "phone": "13800000000",
      "role": 1,
      "status": 1
    }
  }
}
```

## 线索管理（管理员专用）

### 新增线索
- POST `/api/v1/lead`
- 请求参数：
```json
{
  "customer_name": "张三",
  "customer_phone": "13800000000",
  "car_brand": "Toyota",
  "car_model": "Camry",
  "notes": "车况良好",
  "user_id": 1
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1
  }
}
```

### 查询线索列表
- GET `/api/v1/lead?status=待跟进&agent_id=1&page=1&size=20`

### 修改线索状态
- PUT `/api/v1/lead/:id`
- 请求参数：
```json
{
  "status": "已成交",
  "fail_reason": "价格未谈拢"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success"
}
```

### 查看线索详情
- GET `/api/v1/lead/:id`
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "customer_name": "张三",
    "customer_phone": "13800000000",
    "car_brand": "Toyota",
    "car_model": "Camry",
    "notes": "车况良好",
    "user_id": 1,
    "status": "待跟进",
    "fail_reason": null,
    "created_at": "2026-04-21 12:00:00",
    "updated_at": "2026-04-21 12:00:00",
    "last_followup_at": null
  }
}
```

## 结算管理（管理员专用）

### 新增结算
- POST `/api/v1/settlement`
- 请求参数：
```json
{
  "lead_id": 1,
  "profit": 10000,
  "agent_share": 3000,
  "remark": "公司已售出"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1
  }
}
```

### 更新结算状态
- PUT `/api/v1/settlement/:id`
- 请求参数：
```json
{
  "status": "已结算",
  "settled_at": "2026-04-21 12:00:00"
}
```
- 返回：
```json
{
  "code": 0,
  "message": "success"
}
```

### 查询结算列表
- GET `/api/v1/settlement?status=待结算&agent_id=1&page=1&size=20`

### 推送结算提醒
- POST `/api/v1/settlement/:id/notify`
- 功能：触发微信服务通知，推送给对应经纪人
- 返回：
```json
{
  "code": 0,
  "message": "success"
}
```

## 错误码规范
- `code = 0` → 成功
- `code = 1` → 参数错误
- `code = 2` → 鉴权失败
- `code = 3` → 数据不存在
- `code = 4` → 系统错误
